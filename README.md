# 📥 Kafka Consumer Service

A production-ready Kafka Consumer built using Spring Boot and Apache Kafka.  
This service consumes messages from Kafka topics and processes them asynchronously.

---

## 🚀 Tech Stack

- Java 17+
- Spring Boot
- Spring for Apache Kafka
- Maven
- Docker (Optional)

---

## 📌 Features

- Consume messages from Kafka topics
- Consumer group support
- Manual and auto offset commit
- Error handling with retry support
- Configurable via application.yml
- Graceful shutdown
